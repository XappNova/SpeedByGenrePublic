Frequently Asked Questions

Q: Why isn't the speed changing?

A: Make sure the extension is enabled. Also, try refreshing the YouTube page. If the video is an ad, we intentionally do not change the speed.

Q: Can I set a custom speed for a specific channel?

A: Currently, rules are set by category (e.g., Music, Gaming). Channel-specific rules are planned for a future update.
